# Диаграммы поведения системы "Планирование учебного процесса"

## Use Case Диаграмма
Описание основных сценариев использования приложения

### Основные Use Cases:
- Управление расписанием занятий
- Создание и отслеживание заданий
- Работа с учебными проектами
- Система уведомлений и напоминаний

## Диаграммы последовательности

### 1. Создание учебного задания
Показывает процесс создания задания преподавателем с валидацией и настройкой уведомлений

### 2. Отметка выполнения задания
Демонстрирует процесс завершения задания студентом и обновление прогресса проектов

### 3. Просмотр расписания занятий
Отображает механизм загрузки и отображения расписания с проверкой конфликтов

### 4. Ежедневный workflow студента
Комплексный сценарий типичного учебного дня студента

## Используемые инструменты
- **PlantUML** - для кодового представления диаграмм
- **draw.io** - для визуального редактирования
